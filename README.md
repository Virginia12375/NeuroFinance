# NeuroFinance - AI-Powered Financial Platform
A complete financial management platform with AI-powered insights, interactive dashboards, and a futuristic user interface.

## Project Overview
NeuroFinance is a comprehensive financial platform that helps users manage their money intelligently. It features a complete suite of financial tools including transaction tracking, budget management, investment monitoring, savings goals, and account settings - all wrapped in a stunning futuristic UI.

## Tech Stack
HTML5 - Structure

CSS3 - Styling with custom properties

JavaScript - Interactivity & animations

Chart.js & ApexCharts - Data visualization

Three.js - 3D visualizations (main page)

Font Awesome - Icons
